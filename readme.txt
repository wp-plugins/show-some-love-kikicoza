Show some love to the people who make it possible to do what you do.

== Description ==
 
This plugin remembers the details of each person you have added so you don't have to retype all the details in each post and displays it below the relevant posts. We make it easy to show some love.

Major Features include:

* Keeps a database of the providers you previously used
* Links and categories are auto filled
* Add and remove multiple provider with ease
* Automatically appended to content
* Set the show some love header
* Open links in new window
* Custom CSS
* Display links in lists or inline


== Installation ==

Upload the Show Some Love plugin to your blog, Activate it. Under the Settings Menu you will find the Show Some Love option page.

1, 2, 3: You're done!
